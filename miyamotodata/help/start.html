Extracting the Game's Files

To run Miyamoto, you'll need the course_res folder of the game. This is to prove you have a legal copy of the game.

You can use ddd to extract the game's files. A good tutorial is here.

Once you have the games files, open Miyamoto and select the course_res folder. You can now load any level in the game!


Starting Miyamoto!

If you have a precompiled executable, simply double click. Precompiled executables should be available as a Windows exe, a Mac app, or a Linux application made via cx_Freeze, pyinstaller, cython, or nuitka.

For those who cannot run one of the available executable files, or who simply desire not to do so, there is a source distribution available. The source distribution requires the following packages:

  • Python 3.4 or higher
  • PyQt 5.3 or higher

  • Please Note: Linux and Mac users must have WINE installed!

Windows users can get away with downloading the Windows binary installer for Python and the binary Installer for PyQt (first and last links), which will hold your hand and install Qt for you.

Mac OS X Snow Leopard users will need to buildPyQt for both architectures (i386 and x86_64). PyQt will also require a change to the configure.py file - the i386 line should specify x86_64 as well. If you have an older version of Qt, you will need to upgrade to the newest one, and it is strongly recommended you build with the development versions of PyQt.

Linux users must have PyQt5 install via sudo pip3 install pyqt5 and WINE by sudo apt-get install wine for Ubuntu distributions.

If you have a different OS or a non-Ubuntu linux distribution, figure it out yourself.

On the time of writing this, Miyamoto for 32 bit windows will not work, since puzzlehd and gtx_extract are 64 bit. There may be an older build named Miyamoto-SLOW that will work, but is probably very buggy in everything. It is also super slow as the name implies.


Creating New Levels

You still must open aany level in the editor before you can use the menu. To create a new level, simply select "New" from the file menu. A level must have at least one zone and one entrance (ID 0), otherwise the level will not work. You should also add tilesets by clicking "Edit Slot X Tileset").


Opening Existing Files

Miyamoto! has two options for opening files.

"Open Level by Name" produces a list of all levels in NSMBU, and allows you to choose from the list. This option is useful for getting a feel as to how to design your levels, and for looking up references.

"Open Level by File" allows you to select a file from your hard drive and open it. This option will be used to select your works in progress, or to check out the creations of others. It supports both .szs and .sarc filetypes.
